This plugin does one thing - it will show the ViralRank of your blog posts. 

== Description ==

ViralRank™ which is a technology created for http://swarmsports.com can be used anywhere to show how viral a post is. We wrote a blog post announcing the release of this to the world: http://blog.swarmsports.com/2013/07/16/our-new-wordpress-plugin-is-the-tits/ - If you want more awesome ViralRank stuff, sign up for the FREE dashboard - http://viralrank.me/dashboard/ - you can play around with all kinds of cool statistical data there. Enjoy!

== Installation ==

Upload the ViralRank Plugin to your site. Activate. 

== Frequently Asked Questions ==

Q: Will this only show the ViralRank of posts or pages too? 

A: It will show ViralRank of your pages too. 

Q: Is the only thing I need to do is click the activate? 

A: Yes!
